# GWC Root Simulator — 실험 결과

> 13cm 화분(r=6.5cm, h=15cm) 최적 에어룸 배치를 찾기 위한 GA 최적화 실험 이력.

---

## 📋 실험 개요

**목표**: 6개 에어룸을 13cm 화분에 최적으로 배치하여 뿌리 프루닝(분기)을 최대화.

**최종 선정 설계 (V3)**:
```
Mean±Std = 6861 ± 823 (15개 시드 평가)
Surface  = 4657 mm²
Pruning  = 50회 (하단 30 + 중간 20)
Soil Loss = 5.6%
N Uptake = 0.26 mg
Spread   = 9.5%
```

---

## 📊 실험 이력 (V1 → V4)

### V1 — 기준선 (g=1.5, noise=4°, 구 화분)

| 항목 | 값 |
|------|-----|
| 화분 | 12×20cm (구 제품) |
| 중력(g) | 1.5 |
| 1차뿌리 노이즈 | 4° |
| GA | pop=20, gen=12 |
| 평가 | 싱글시드 (버그) |
| 점수 | 8764 (단일시드, 신뢰도 낮음) |
| 프루닝 구역 | 하단 20 / 중간 0 / 상단 0 |

→ 구 화분 기준 첫 번째 최적화. 싱글시드 버그로 신뢰도 낮음.
→ **문제 발견**: 모든 프루닝이 하단에만 집중됨.

### V2 — 중력 미조정 (g=1.3)

| 항목 | 값 |
|------|-----|
| 화분 | 6.5×15cm (신 제품) |
| 중력(g) | 1.3 |
| 1차뿌리 노이즈 | 4° |
| GA | pop=25, gen=15, elite=0.15, mut=0.8 |
| 평가 | 15시드 |
| 점수 | **6968±825** |
| 프루닝 구역 | 하단 35 / 중간 0 / 상단 0 |

→ g를 1.5→1.3으로 낮췄지만 프루닝 구역 분산 실패.
→ **가설**: 중력만 낮춰서는 부족. 뿌리가 더 퍼져야 함.

### V3 ✅ — 중력 완화 + 노이즈 증가 (g=1.0, noise=8°)

| 항목 | 값 |
|------|-----|
| 화분 | 6.5×15cm |
| 중력(g) | **1.0** |
| 1차뿌리 노이즈 | **8°** |
| GA | pop=25, gen=15, elite=0.15, mut=0.8 |
| 평가 | **15시드** |
| 점수 | **6861±823** |
| 프루닝 구역 | ✅ **하단 30 / 중간 20 / 상단 0** |

| 시드별 점수 |
|------------|
| 7975, 7287, 8089, 7253, 5880, 7820, 5812, 6090, 6176, 6698, 6447, 7075, 5431, 7267, 7617 |

→ **noise=8°가 결정적.** 1차뿌리가 ±8°로 퍼지면서 중간 높이(z=5~7cm)의 에어룸 도달 성공.
→ 프루닝이 하단(30) + 중간(20)으로 분산됨.
→ **최종 선정.**

### V4 — 과도한 완화 (g=0.8, noise=10°)

| 항목 | 값 |
|------|-----|
| 중력(g) | 0.8 |
| 1차뿌리 노이즈 | 10° |
| 점수 | 6503±645 |
| 프루닝 구역 | ❌ 하단 19 / 중간 6 / 상단 0 |

→ **퇴행.** 뿌리가 너무 퍼져서 GA가 방어적으로 모든 에어룸을 하단에 몰아넣음.
→ 핵심 통찰: "너무 퍼져도 안 된다 — 최적의 중력-노이즈 균형이 존재함."

## 💡 실험을 통해 배운 점

### 1. 중력-노이즈 균형이 핵심

| 조합 | 결과 |
|------|------|
| g=1.5, noise=4° | 뿌리가 직하, 프루닝 전부 하단 |
| g=1.3, noise=4° | 거의 변화 없음 |
| **g=1.0, noise=8°** ✅ | **프루닝 하단+중간 분산 성공** |
| g=0.8, noise=10° | 너무 퍼짐, GA가 하단으로 회귀 |

→ **g=1.0 + noise=8°가 최적 균형점.**

### 2. 15시드 평가의 중요성

단일시드 점수와 15시드 평균의 차이가 최대 20% 이상 발생.
→ **설계의 진짜 성능을 알려면 최소 5개, 바람직하게는 10~15개 시드로 평가해야 함.**

### 3. 상단(U) 프루닝 — 남은 과제

현재 최적 설계에서 z=12cm 상단의 에어룸(④)은 뿌리가 도달하지 못함.
- 도달하려면 2~3세대 분기가 더 넓게 퍼져야 함
- 현재는 2세대 분기 후에도 중력에 의해 아래로 성장
- 해결 방안: `h`(hydrotropism) 활성화 + diffusion의 gravity_bias 조정, 또는 2세대 노이즈 증가
